# 主题:github(克制专业)

继承 `PRINCIPLES.core.md`,下面是本风格特有的补充。数值见同目录 `tokens.css`。

## 风格性格
- 近乎黑白灰,蓝色 = 链接 / 可点,绿色 = 主操作(每屏最多一个主按钮)。
- 区域靠 1px 浅灰边框分隔,**不靠阴影、不靠彩色背景**。
- 系统字体栈,无需加载;圆角小(卡片 6px、小元素 4px)。
- 整体偏紧凑、信息密度高。

## 颜色什么时候能用(速查)
- 主操作按钮 → 绿 `--color-success-emphasis`
- 链接 / 可点文字 → 蓝 `--color-accent-fg`
- 状态:超期→红,临近→黄,阻塞/停滞→紫,正常/完成→绿或灰
- 状态表达:**小圆点** 或 **小徽标(subtle 底)**,二选一,别又点又块
- ❌ 不允许:彩色卡片背景、彩色大区块、用颜色区分布局栏目、装饰性渐变

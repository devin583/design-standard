# screenshots/ —— 视觉参照基准

AI 对「克制 / 精致 / GitHub-like」的理解容易飘。放几张**真实的参照截图**当锚点,
偏差会明显变小。改 UI 前可让 AI 先看这里对齐目标观感。

## 怎么放
按主题分目录,每张配一个简短文件名说明它示范了什么:
```
screenshots/
  github/
    issue-list.png       # 高密度列表 + 克制用色的范例
    settings-page.png    # 设置页信息架构范例
    detail-modal.png     # 弹窗/详情范例
```

## 建议来源
- 你认可的真实产品截图(GitHub / Linear 等),或
- 你自己项目里做得好的页面截图(也可在 HARVEST 提炼时一并收集)。

> 注:本目录现在是占位说明,放进截图后删掉本说明的占位性质即可。
